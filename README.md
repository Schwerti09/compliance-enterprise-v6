# ComplianceVault Enterprise Machine v6

Full scale Next.js App Router project.

## Structure
- `app/(dashboard)`: Protected App Area with Sidebar Layout
- `lib/store.ts`: Mock Database
- `components/ui`: Full UI Kit (Table, Badge, Input, Card...)

## Setup
1. `npm install`
2. `.env.local` (GEMINI_API_KEY)
3. `npm run dev`
